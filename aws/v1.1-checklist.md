## Troubleshooting

### If solve crashes (OOM):
1. Check memory usage: `free -h` (inside EC2)
2. Try larger instance: r6a.4xlarge (128 GB)
3. Verify 16-bit compression is enabled

### If compile fails:
1. Check Rust version: `rustc --version`
2. Update if needed: `rustup update`

### If S3 upload fails:
1. Check IAM role: `aws sts get-caller-identity`
2. Verify permissions on bucket

### If SSH times out:
1. Check security group allows your IP
2. Update IP if changed: `aws ec2 authorize-security-group-ingress ...`
